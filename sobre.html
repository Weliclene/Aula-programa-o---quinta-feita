Sobre o Site de Notícias

Este site foi criado para fornecer as últimas notícias do mundo de forma rápida e fácil. Nosso objetivo é manter os usuários atualizados com as informações mais recentes sobre os principais eventos que acontecem ao redor do mundo.

O site foi desenvolvido como parte de um projeto acadêmico para a faculdade, usando tecnologias como HTML e CSS.
